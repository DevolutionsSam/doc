# {{ fr.TOPICTITLE }} 
Pour protéger vos informations, nous utilisons un fournisseur tiers de service d&apos;échange de fichiers sécurisé, à savoir Sharefile.  

Pour nous envoyer un fichier, il suffit de cliquer sur [Sharefile](https://devolutions.sharefile.com/filedrop), d&apos;inscrire vos informations et il vous sera possible de télécharger un ou plusieurs fichiers. Ceux-ci ne sont pas accessibles par tous les employés de Devolutions, mais seulement par ceux de l&apos;équipe du Centre de services.
