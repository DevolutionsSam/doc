## Configuration minimale 

* Système d&apos;exploitation : macOS Catalina (10.15) 
* Processeur : 1 GHz 
* Mémoire vive (RAM) : 4 Go 
* Résolution d&apos;écran : 1440 x 900 px 
* Espace de disque dur : 1,5 Go
