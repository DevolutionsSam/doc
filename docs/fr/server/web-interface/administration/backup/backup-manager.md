title: Gestionnaire de sauvegardes
---
La section ***Gestionnaire de sauvegardes*** permet aux administrateurs de configurer les paramètres de sauvegarde de la base de données et du dossier de l&apos;application Web.  
![Administration - Gestionnaire de sauvegardes](/img/fr/server/clip10403.png)

	<tr>
		<td>
Chemin du fichier de backup 
		</td>
		<td>

Le chemin d&apos;accès au dossier dans lequel la sauvegarde de la base de données SQL sera enregistrée. Nous vous recommandons d&apos;utiliser un dossier réseau partagé avec des autorisations appropriées définies dessus pour pouvoir enregistrer le fichier de sauvegarde.  

Remarque : Comme la commande de sauvegarde s&apos;exécute sur le serveur SQL et la machine où {{ fr.DPS }} est installé, ce chemin doit être accessible à partir de ce serveur SQL et depuis le serveur où {{ fr.DPS }} est installé. Nous suggérons de configurer le chemin de backup de la base de données avec un chemin UNC. 
		</td>
	</tr>

		<td>
